# Inventory Endpoints

This module handles material, equipment, and resource management within a business.

## Endpoints

- `GET /api/v1/inventory/items/`: List all inventory items (materials, equipment).
- `POST /api/v1/inventory/items/`: Create a new inventory item.
- `GET /api/v1/inventory/transactions/`: List inventory transactions (in, out, adjustment).
- `POST /api/v1/inventory/transactions/`: Record a new inventory transaction.
- `GET /api/v1/inventory/requests/`: List material requests from field users.
