# Config Endpoints Documentation

This directory serves as the root URL configuration module for the entire backend application.

## Endpoints

*   `/admin/`: The built-in Django administration panel for managing application data models.
*   `/api/auth/`: Delegates authentication, user, and role management endpoints to the `authentication` app.
*   `/api/businesses/`: Delegates core business logic, schema setups, and user assignments to the `business_meta` app.
*   `/api/relations/`: Delegates dynamic relation definitions and mapping to the `business_meta` app.
*   `/api/`: A fallback root API path delegating general operations (e.g., global inventory items) to the `inventory` app.
*   `/api/schema/`: Serves the raw OpenAPI 3.0 schema in YAML/JSON format generated by `drf-spectacular`.
*   `/api/docs/`: Serves the interactive Swagger UI interface to browse and test the OpenAPI schema.
*   `/api/redoc/`: Serves the interactive ReDoc UI interface to browse and test the OpenAPI schema.
