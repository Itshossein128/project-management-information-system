# Frontend Routes

This document outlines the main routes configured for the frontend application.

## Main Routes

- `/`: The landing or welcome page.
- `/login`: The user authentication interface.
- `/dashboard`: The main overview page after successful login.
- `/businesses`: List of all businesses the user has access to.
- `/businesses/:id`: Detailed view of a specific business dashboard.
- `/businesses/:id/warehouse`: Interface for inventory and resource management within a business.
- `/settings`: User or application settings view.
